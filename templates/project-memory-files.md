# AI Web File Layout Template

```text
AI Web/
  Core/{PROFILE,PREFERENCES,GOALS,DECISIONS,CHANGELOG}.md
  Projects/<area>/{INDEX,CURRENT-STATE}.md
  Projects/<area>/Objects/
  Inbox/
  Archive/
  Templates/
```

Allowed area slugs: `faculdade`, `pessoal-e-carreira`, `organizacao-e-decisoes`, and `design-e-fotografia`. Keep one authoritative `CURRENT-STATE.md` per area. Use Inbox filenames `YYYY-MM-DD-HHMM-<project>-<topic>.md` and retain source/date through Archive.
