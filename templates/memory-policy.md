# Web Memory Policy Template

Store only concise, durable, reviewed Web preferences, goals, decisions, state, learnings, and policies. Each record carries scope, type, date, source, confidence, and status when relevant.

Candidates start pending. Consolidate confirmed non-conflicting content only; hold conflicts for user approval; archive every processed candidate. Exclude credentials, personal-sensitive content, raw chats, temporary notes, development state, and CLI technical memory.
