# Contract: ChatGPT Project Instructions

Every Web Project instruction MUST:

1. State that it belongs to the personal/executive Web environment.
2. Limit ChatGPT Memory to small global preferences.
3. Name the Core files, area index, and current state as mandatory-first reading.
4. Prohibit technical repository context, development logs, credentials, and CLI memory.
5. Define durable versus temporary information.
6. On session close, create a structured candidate and never silently consolidate memory.
7. State that any app write needs explicit user confirmation; otherwise request manual
   save to the local Drive clone.

The four area variants differ only by their area slug and relevant object folders.
