# Permissions and Approval

| Action | Owner | Requirement |
| --- | --- | --- |
| Read synchronized Drive content | User or ChatGPT, when available | Verify scope and synchronization state. |
| Write through a ChatGPT app | User-directed ChatGPT action | Explicit confirmation immediately before the write. |
| Manual local save | User | Save candidate to the authorized synchronized clone and verify sync. |
| Consolidate a conflict | User | Explicit approval. |
| Route technical memory | CLI owner | Keep out of Web folders and Projects. |

If an app cannot write, does not show the expected account, is still indexing, or reports a conflict, do not imply success. Produce a candidate and use the manual path.
