# Web Workflow

Use ChatGPT Projects for area-scoped instructions, ChatGPT Memory only for small global preferences, and the `AI Web` Drive hierarchy for durable structured records.

Before relevant work, read Core records followed by the selected area's `INDEX.md` and `CURRENT-STATE.md`. Use the Google Drive app with sync for discovery only when it is available and synchronized. If it is unavailable, indexing, disconnected, or lacks a confirmed write action, ChatGPT produces a candidate for the user to save manually to the synchronized local clone.

Direct app writes require explicit user confirmation for each action. A synchronized or indexed file is not proof of an authorized source-of-truth write. See [Project instructions](../web/project-instructions.md), [session closure](../web/session-closure.md), and [curation](../web/curation.md).
