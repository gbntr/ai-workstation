# CLI Workflow Boundary

CLI technical memory is an independent private repository. It stores approved technical state, implementation evidence, project records, and CLI agent context under its own review process.

Do not place CLI memory, development logs, code state, credentials, or private project details in ChatGPT Web Projects or the `AI Web` hierarchy. Conversely, do not copy Web/Drive personal memory into the private technical-memory system. Route uncertain material to the owner for classification.
