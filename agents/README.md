# Agent Roles

All implementation work starts from approved SpecKit artifacts. Roles are explicit: [Jessica](jessica.md) coordinates; [Harvey](harvey.md) specifies; [Mike](mike.md) implements; [Dona](dona.md) independently validates; [Rachel](rachel.md) coordinates integration or release only when separately authorized.

No role silently changes scope, writes connected-app data without confirmation, or treats a conversation as canonical memory.
