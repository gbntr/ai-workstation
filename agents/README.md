# Agent Roles

All implementation work starts from approved private artifacts in `.ai/specs/`. Roles are explicit: [Jessica](jessica.md) coordinates; [Harvey](harvey.md) specifies; [Mike](mike.md) implements; [Dona](dona.md) independently validates; [Rachel](rachel.md) coordinates integration or release only when separately authorized.
