# ai-workstation

Public, human-friendly documentation for how I use ChatGPT web and the terminal-side agent workflow.

## What lives here

- `web/` — ChatGPT web: Memory, Projects, Work, and Drive rules.
- `cli/` — terminal workflow, development flow, and technical operations.
- `agents/` — agent roles, responsibilities, and routing.
- `docs/` — canonical navigation and operating docs.
- `.ai/` — private agent-only artifacts; ignored by Git.

## Design priorities

1. Human-friendly.
2. Minimal leftover structure.
3. Clear separation between web and CLI.
4. Easy manual upload/recreation later.

## Language

Tracked docs are written in English for reuse and public readability.

## Start here

1. Read `docs/index.md`.
2. Then open `web/README.md` or `cli/README.md` depending on the workflow.
